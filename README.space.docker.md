# Kizuna Voice Studio

This Space deploys a preview-focused version of Kizuna Voice Studio.

- It generates seed voice previews from Japanese instructions.
- It is configured for the current `cpu-basic` plan.
- Paid GPU hardware is not available on the current organization plan, so full training is not exposed here.

For the full workflow, use the desktop app or another runtime with paid GPU support.
